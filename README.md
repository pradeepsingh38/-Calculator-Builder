Drag & Drop Calculator Builder

Description

This project is a dynamic, interactive calculator builder that allows users to drag and drop components to create a custom calculator. Built using React, Zustand, and Tailwind CSS, it supports features like dark mode, undo/redo functionality, and state persistence with local storage.

Features

Drag & Drop Functionality to build custom calculators.

Dark Mode Toggle for better accessibility.

Undo/Redo actions to correct mistakes.

State Persistence using Local Storage.

Responsive Design for mobile and desktop users.
